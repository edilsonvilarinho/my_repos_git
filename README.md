# my_repos_git

Projeto que consome API https://api.github.com/users/{user}/repos

 - MVC
 - Provider
 
# Estrutura do projeto

  - Padrão MVC
    - Funcionalidade 
      - controller
      - model
      - imagens
      - service
      - view
   - Provider como controle de estado concentrado no AppWidget para ser utilizado em qualquer parte do app pelo o comsumer.
